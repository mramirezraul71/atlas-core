# Asset History Profiles

Esta carpeta define perfiles condicionados que Atlas puede usar para comparar el
escenario actual con familias historicas recurrentes.

Perfiles incluidos:
- `trend_follow_profile.md`
- `range_response_profile.md`
- `earnings_gap_profile.md`

Uso:
- seleccionar el perfil por regimen y tipo de evento
- comparar la sesion actual con la conducta historica esperada
- degradar o bloquear si el activo se comporta fuera de su familia historica
