# Decision Gate

## Principio
Atlas no debe evaluar señal ni ejecutar si el contexto no es suficientemente claro.

## Permitir
- Regimen identificable
- Liquidez aceptable
- Sesion actual alineada
- Riesgo macro controlado
- Evidencia historica razonable

## Degradar
- Contexto util pero ambiguo
- Evento cercano sin confirmacion adicional
- Premarket o intradia sin aceptacion limpia
- Calidad estadistica incompleta

## Bloquear
- Riesgo extremo
- Liquidez baja
- Evento macro de alto impacto con claridad insuficiente
- Comportamiento del activo atipico respecto a su perfil
- Contradiccion fuerte entre HTF, sesion y señal

## Reglas De Salida
- Toda decision debe incluir confianza.
- Toda decision debe incluir razones.
- Toda decision debe incluir fuentes o trazas.
